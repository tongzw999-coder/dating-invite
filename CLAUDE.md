# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 提供在此代码库中工作的指导。

## 项目概述

单文件约会邀请网页（`index.html`）—— 一个有趣的多步骤表单，流程包括：

- 浮动爱心背景动画
- "不要"按钮会逃跑（鼠标悬停/点击时躲避）
- 多步骤流程：名字 → 时间 → 场地 → 美食 → 结果
- 页面切换时的闪光效果
- 支持移动端触摸操作

## 开发

这是静态 HTML 文件，无需构建工具或测试。直接在浏览器中打开 `index.html` 即可预览修改效果。

## 代码架构

- 单 HTML 文件，CSS 和 JavaScript 内联
- 页面切换通过 CSS 类（`.active`）和原生 JS 实现
- 游戏状态存储在简单的 `selections` 对象中
- 浮动爱心通过 JavaScript 动态创建
